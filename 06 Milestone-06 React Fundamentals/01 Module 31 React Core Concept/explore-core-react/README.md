# Module 31: React Core Concepts (Part 1)

This module covers the initial core concepts of React, focusing on components, props, and conditional rendering.

## Key Topics Practiced:

- **JSX**: Understanding JavaScript XML and how to write HTML-like code in React.
- **Components**: Creating functional components like `Student`, `Developer`, `Player`, and `Salami`.
- **Props**: Passing data from parent to child components using `props` and destructuring.
- **Dynamic Content**: Rendering dynamic values and expressions within JSX using curly braces `{}`.
- **Conditional Rendering**: Practiced various ways to render content conditionally in the `Todo` component:
  - If-else statements.
  - Ternary operator (`condition ? true : false`).
  - Logical AND (`&&`) operator.
  - Logical OR (`||`) operator.
- **Mapping Data**: Rendering lists of data using the `.map()` function (e.g., `actors` array and `singers` objects).
- **Styling**: Applying inline styles and using CSS classes in React components.

## Components Created:

- `App.jsx`: Main entry point to demonstrate all concepts.
- `Todo.jsx`: Focused on conditional rendering techniques.
- `Actor.jsx`: Demonstrates passing simple string props.
- `Singer.jsx`: Demonstrates passing object data as props.
- `Student`, `Developer`, `Player`, `Salami`: Experimental components for learning props and styling.
